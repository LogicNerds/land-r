Quick Setup Guide

Welcome! Follow these simple steps to get your Land-r up and running in minutes.
🎥 Visual learner? Check out our quick setup video!

Step 1: Add Your Business Info

Fill in your company name, contact email, and phone number on this page.

Tip: Upload a logo to make your Land-r look more professional!

Step 2: Add Your Products & Services

Go to "My Products & Services" and add the items customers can request.

Tip: Use the "Show Price" toggle on each service to control whether pricing displays on your Land-r.

Step 3: Customize Your Land-r

Visit "My Land-r" to customize the welcome message and colors.

Tip: A clear headline and friendly welcome message help customers feel confident submitting a request.

Step 4: Share Your Land-r

Copy your Land-r link and share it — on social media, in texts, or embed it on your website.

Tip: Add it to your social media bio, Google Business profile, or email signature.

Step 5: Manage Your Requests

Check "My Requests" to see incoming requests. Use the stage tracker to move each one through your process.

Tip: New requests stay in the "New" tab until you change their status — so nothing slips through!

That's it! You're ready to start collecting requests. Questions? Feel free to explore the interface — everything is designed to be intuitive and easy to use.

Where to Install Your Pop-up Widget

Quick Answer: Paste the code right before the closing </body> tag on your website.

WordPress

Option 1: Use a plugin like "Insert Headers and Footers"

  • Install the plugin from WordPress admin
  • Go to Settings → Insert Headers and Footers
  • Paste code in "Scripts in Footer" section

Option 2: Edit theme files directly

  • Appearance → Theme Editor
  • Open footer.php
  • Paste code before </body>

Wix

  • Settings → Custom Code
  • Click "Add Custom Code"
  • Paste your code
  • Select "Body - end"
  • Apply to all pages

Squarespace

  • Settings → Advanced → Code Injection
  • Paste code in "Footer" section
  • Click Save

Shopify

  • Online Store → Themes
  • Click "Actions" → "Edit Code"
  • Open theme.liquid file
  • Paste code before </body>
  • Click Save

Weebly

  • Settings → SEO
  • Scroll to "Footer Code"
  • Paste your code
  • Click Save

Custom HTML Site

  • Open your HTML file in a text editor
  • Find the </body> tag (near the bottom)
  • Paste code right before it
  • Save and upload to your server
Need help? Contact your website developer or hosting support — they can install this in 2 minutes!
